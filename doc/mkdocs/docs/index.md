# Welcome to Pi4U

Pi4U is a HPC framework for Bayesian uncertainty quantification of large scale computational models.

# Modules and algorithms

## Inference
  TMCMC
  ABC-Subsim
  DRAM

## Single-objective optimization
  CMAES
  Amalgam

## Multi-objective optimization
  NSGA-II
  MOCMAES
  MOEAD

# Implementations
  Parallel  
  Sequential
